# 🎓 Plataforma de Inscripción a Talleres

Página web que permite a usuarios inscribirse en talleres y cursos disponibles.

## Stack tecnológico
- **Frontend:** React + Vite
- **Backend:** Node.js + Express
- **Base de datos:** PostgreSQL
- **ORM:** Sequelize
- **Autenticación:** JWT

## Alcance del proyecto
- Gestión de talleres/cursos disponibles
- Inscripción de alumnos a talleres
- Autenticación de usuarios
